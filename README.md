# homebrew-alochi

Alochi Monitoring macOS ilovasi uchun rasmiy Homebrew tap.

## O'rnatish

```bash
brew tap rusthype/alochi
brew install --cask alochi-monitoring
```

## Yangilash

```bash
brew upgrade --cask alochi-monitoring
```

Bu tap `rusthype/alochi-monitoring` repozitoriyasidagi GitHub Actions workflow
(`.github/workflows/macos_release.yml`) orqali har bir `vX.Y.Z` tegida
avtomatik yangilanadi.
